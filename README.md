# T test简写 T.js主要是总结一些常用的验证函数

统一返回Boolean值

## install

```
npm install t
```

## API

#### 是否是字符串
```
T.isString(str)
```

#### 是否是数字
```
T.isNumber(str)
```

#### 是否是undefined
```
T.isUndefined(str)
```

#### 是否是数组
```
T.isArray(str)
```

#### 是否是对象
```
T.isObject(str)
```

#### 是否是日期
```
T.isDate(str)
```

#### 是否是FormData
```
T.isFormData(str)
```

#### 是否是File
```
T.isFile(str)
```

#### 是否是URLSearchParams
```
T.isURLSearchParams(str)
```
